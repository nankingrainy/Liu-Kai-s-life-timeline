# life-timeline

·本人籍贯山西省xx市xx县，祖上三代贫农，耕作于村，并无家业。在父亲的努力下，终于在县城安家立业，并于2001年在县医院诞下一子，取名为刘x。在该时间线内，隐去名讳，以作者笔名王若斯称呼。

·王若斯在他3岁的时候去了幼儿园，名为新华幼儿园，三年期间并未取得一朵好孩子才有的小红花，毕业时老师勉为其难颁发一朵，以消遗憾。

·在王若斯6岁的时候，就读于黄河路小学。在小小的王若斯眼中，这所小学大大的，有可以肆意奔跑的前后广场，有高不可攀的巨大柳树，以及宛如迷宫相互连通的三栋教学楼。在这里，王若斯度过了他平凡且快乐的童年时光，有赢不玩的圆片、奢望的塞尔号nono以及和好朋友玩的《造梦西游》。当然，平凡是当时的感受，快乐是成人后的感受。

·2009年，家庭诞下一女，自此王若斯解锁了一个新Title——哥哥。

·在王若斯12岁的时候，他小学毕业了，并于当年暑假举办12岁圆锁礼。当时有一好朋友，在小区院内饲养一窝小兔，王若斯很是喜欢，朋友答应送他一只作为生日礼物。不过若斯没能等到生日那天，偷偷去兔笼拎走一只。后被对方家母发现，大笑，解释想送王若斯的是另外一只，这只要留着生崽，双方互换小兔。

·小升初的各科成绩已不可考，最终被县公立红星中学录取，就读于三班，其中班主任为赵白娥老师，数学郭爱民老师。自此王若斯正式变成了一名中学生

# The Life Timeline

- 作者籍贯山西省xx市xx县，祖上三代贫农，耕作于村，并无家业。在父亲的努力下，终于在县城安家立业，并于2001.6.21于县医院诞下一子，取名为刘x。

- 作者在他3岁的时候去了幼儿园，名为xx幼儿园，三年期间并未取得一朵好孩子才有的小红花，毕业时老师勉为其难颁发一朵，以消遗憾。

- 在作者6岁的时候，就读于xxx小学。在小小的作者眼中，这所小学大大的，有可以肆意奔跑的前后广场，有高不可攀的巨大柳树，以及宛如迷宫相互连通的三栋教学楼。在这里，作者度过了他无聊且快乐的童年时光，有赢不玩的圆片、奢望的塞尔号nono以及和好朋友玩的《造梦西游》。当然，无聊是当时的感受，快乐是成人后的感受。

- 2009年，家庭诞下一女，自此作者解锁了一个新Title——哥哥。

- 2012年，在作者12岁的时候小学毕业了，并于当年暑假举办12岁圆锁礼。当时有一好朋友，在小区院内饲养一窝小兔，作者很是喜欢，朋友答应送他一只作为生日礼物。不过作者没能等到生日那天，偷偷去兔笼拎走一只。后被对方家母发现，大笑，解释想送作者的是另外一只，这只要留着生崽，双方互换小兔。

- 作者小升初的各科成绩已不可考，不过成绩还不错，最终被县公立xx中学录取，就读于三班，其中班主任为赵xx老师，数学郭xx老师。自此作者正式变成了一名中学生，开启了人生新阶段。

- 在新班级里，作者的成绩并不很好，位于20/50左右位次。不过班里有很多教师子弟，比如班主任的女儿和外甥，数学老师的儿子等，在当时的作者眼里，有一种关系户班级的神奇感觉，后来感觉这只是小县城的缩影罢了。

- 2013年，初一上半学期，作者喜欢上了班级的女同学，开始暗恋。

- 2015年，初二结束后的暑假，作者表白，对方答曰：你终于表白了，于是作者开始了初恋。依稀记得那是一个8月。

- 2016年，作者初中毕业，成绩不错，顺利被市重点中学录取。不过成绩更好的女朋友反而留在县里读书，从此分手。

- 2016年9月，作者就读于XX市第一中学校，成绩倒数，原来初中那点引以为傲的成绩如此地不够看，遂自卑。

- 2019年6月，作者顺利读完高中，高考成绩556分，虽远不及预期，但完全属于作者实力，最终被杭州电子科技大学录取，于当年9月前往杭州读书，从此开始了土狗进城的故事。
